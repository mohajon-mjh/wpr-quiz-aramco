# WPR Quiz MJH

A computer-based test (CBT) style practice quiz app for work permit receiver (WPR) certification preparation.

## Features
- 350 practice questions across 7 categories (Work Permit Basics, Safety Rules, PPE, Permit Types, Emergency, Hot Work, Confined Space)
- Instant right/wrong feedback with auto-advance
- Timed test mode with progress tracking
- Reference Guide: common abbreviations, permit-type quick reference, permit color coding, and a visual safety-items checklist
- Installable as a Progressive Web App (PWA)

## Tech Stack
- React + Vite
- Deployed via GitHub Pages (`/docs` folder)
